# GameHub - 欧美流行游戏平台

一个现代化的游戏网站，采用黑色主题设计，提供50个精选游戏，支持游戏分类、搜索和详情页面。

## 🎮 功能特点

### 首页功能
- **英雄区域**: 展示精选游戏，包含可玩的iframe游戏
- **游戏网格**: 5列布局，展示50个游戏
- **分类过滤**: 支持按游戏类型筛选（全部、动作、策略、益智、冒险）
- **搜索功能**: 实时搜索游戏名称、分类和描述
- **响应式设计**: 适配各种设备屏幕

### 游戏详情页面
- **游戏信息展示**: 完整的游戏信息、评分和描述
- **iframe游戏**: 直接在页面中玩游戏
- **全屏模式**: 支持游戏全屏显示
- **相关推荐**: 智能推荐相似游戏
- **收藏功能**: 收藏喜欢的游戏
- **分享功能**: 分享游戏链接

### 设计特色
- **黑色主题**: 现代化的深色设计
- **霓虹绿色**: 醒目的绿色作为主色调
- **渐变效果**: 丰富的渐变和动画效果
- **玻璃拟态**: 半透明背景和模糊效果
- **悬停动画**: 丰富的交互动画

## 🚀 快速开始

### 文件结构
```
GameHub/
├── index.html          # 首页
├── game-detail.html    # 游戏详情页面
├── styles.css          # 主样式文件
├── detail-styles.css   # 详情页面样式
├── script.js           # 首页JavaScript
├── detail-script.js    # 详情页面JavaScript
└── README.md           # 项目说明
```

### 使用方法
1. 下载所有文件到本地目录
2. 用浏览器打开 `index.html` 即可开始使用
3. 点击任意游戏卡片进入详情页面
4. 在详情页面可以直接玩游戏

## 🎨 技术特性

### 前端技术
- **HTML5**: 语义化标签和现代HTML特性
- **CSS3**: Flexbox、Grid、动画、渐变
- **JavaScript**: ES6+语法，模块化编程

## 📱 响应式设计

网站完全响应式，支持以下设备：

- **桌面端**: 1200px+ (5列游戏网格)
- **平板端**: 768px-1024px (4列游戏网格)
- **手机端**: 480px-768px (3列游戏网格)
- **小屏手机**: <480px (2列游戏网格)

## 🔧 自定义配置

### 添加新游戏
在 `script.js` 和 `detail-script.js` 中的 `gamesData` 数组添加新游戏：

```javascript
{
    id: 51,
    title: "新游戏名称",
    category: "游戏分类",
    rating: 4.5,
    image: "游戏图片URL",
    iframe: "游戏iframe URL",
    description: "游戏描述"
}
```

### 修改主题颜色
在 `styles.css` 中修改CSS变量：

```css
:root {
    --primary-color: #00ff88;
    --background-dark: #0a0a0a;
    --background-light: #1a1a1a;
    --text-color: #ffffff;
}
```

### 调整布局
- 修改游戏网格列数: 更改 `.games-grid` 的 `grid-template-columns`
- 调整游戏卡片大小: 修改 `.game-card` 的尺寸
- 更改英雄区域布局: 调整 `.hero-section` 的 `grid-template-columns`
